## Propósito del Repositorio
Este repositorio contiene la **Entrega #1: Hola Mundo**, correspondiente a la investigación del lenguaje de programación **Julia**. El objetivo principal es explorar las características, sintaxis básica, tipos de datos
y estructuras de este lenguaje para compararlo con otros paradigmas de programación y ampliar nuestra perspectiva en la resolución de problemas algorítmicos.
